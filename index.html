📖 Tentang Media Pembelajaran Ini

Media ini dirancang untuk membantu siswa memahami spektrum gelombang elektromagnetik melalui visualisasi interaktif. Anda dapat mengeksplorasi setiap jenis spektrum (Radio, Microwave, Inframerah, Cahaya Tampak, Ultraviolet, Sinar-X, dan Sinar Gamma), melihat bagaimana panjang gelombang, frekuensi, dan energi saling berkaitan, serta mempelajari aplikasi praktis dari setiap spektrum.

Tidak memerlukan instalasi • Berjalan offline • Responsive untuk semua perangkat
